    Zone cleanup

    Reset a zone when a recorder bug contaminated its data (wrong floor counts, ghost cells, garbage at world origin, etc.). Quarantines every dump that contributed, deletes the merged JSON, and triggers a re-merge. Fresh recordings rebuild it cleanly.

    Quarantine

    Dumps that have been moved out of the merge pipeline. Use the buttons in the table above (or "Reset zone") to add to this list. To restore a quarantined dump, move the file back into dumps/ on the server and trigger a re-merge.
